Task 7 - Timing

Javascript is able to control the timing of functions. There are 3 key ways to do this

setTimer()
Delays the code for a specified time
setInterval()
Continously calls a function a specified period of time apart
requestAnimationFrame()
Similar to setInterval() but with more control

Create a page that scrolls automatically through different images - ie create a carousel.

There are several strategies you can take to solve this problem.

What you CANT do...
Copy code from last year
Use examples that use JQuery or other frameworks
What you COULD do...
Name your images sequentially and use a loop counter as part of the filename
Use an IF statement or a switch statement on the counter to change the image name
Load the image names into an array and use the counter to select an item from the array (advanced)
Write a function to increment the array number and update the image displayed
- Use the setInterval() function to loop and call your image-swap function
