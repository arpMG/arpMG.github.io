Activity - Statistics Calculator

Create a page that:

  • Asks the user to enter numbers until they enter a blank
  • Once they enter a blank (ie no value), the program returns back some statistics about the data - see below
  • If their value is not a number, a warning is given, and that value is ignored

Statistics to display:

  • Count of entries
  • Minimum of entries
  • Maximum of entries
  • Range of entries
  • Sum of entries
  • Average (mean) of entries
  • Output your answers in a table (hard)
  • Mode and Median of entries (harder - you will need to use arrays)
  • Standard Deviation (hardest - do your research)

Hints

  • Start with the just tracking the count. Get your loop right.
  • Then add some variables to keep track of the other numbers.

  • A separate folder for your JS folio work
  • An images folder - make sure you rename any borrowed images to useful filenames
