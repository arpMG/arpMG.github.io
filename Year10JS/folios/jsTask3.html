Task 3 - Using If Statements

An if statement is an example of a SELECTION statement.
It allows control of the program to branch into different code, depending on certain conditions.

JavaScript

In JavaScript an if statement has the syntax

if(condition){

...code to run if condition is true

}else{

...code to run if condition is false

}

Activity - Selection Statements

Create a page that:

  1. Asks the user for their age in years and tells them if they are old enough to get their Learner's Permit or not.
  2. Asks the user for a character and tells them if it is a vowell or a consonant or neither.

Extension: Have the use enter their date of birth and calculate their age from that, then tell them if (or when) they can get their Learner's.
