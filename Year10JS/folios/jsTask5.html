Task 5 - Using Loops (For Loop)

A loop, or repetition structure, allows sections of code to run multiple times.

There are 3 types of loops, which are used depending on your circumstances

Fixed Count Loop - FOR/NEXT
Used when you know the number of times you want to loop
There is also a FOREACH loop used loop through arrays and other collections - more on that another time
Check first loop - WHILE
Check the run condition before first time
Might never run
Check last loop - DO/WHILE
Runs once first, THEN checks if to run it again

JavaScript

In JavaScript a Loop has the syntax

FOR/NEXT LOOP (fixed count)
for(settings to start ; condition to end ; things to do at end of each loop){
 code to run inside each loop
}
WHILE LOOP (check first)
while(condition is true){
 code to run inside each loop
}
DO/WHILE LOOP (check last)
do{
 code to run inside each loop
}while(condition is true){
