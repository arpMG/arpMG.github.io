Task 4 - Using Functions with parameters

A function is a small sub-program which performs a specific task and can return the result of some processing to the main program

A parameter is a variable that is passed into a function. The value that is passed is used inside the function to complete its task.

Activity - Simple Parameter Activities

Create a page that has the following.

  • A function called sum() which takes two numbers as parameters and returns the sum of those two numbers to the calling function.
  • A function called diff() which takes two numbers as parameters and returns the difference between them (as a positive number) to the calling function.
  • A function called in_out() which collects user input, passes these to the sum() and diff() functions, then outputs the results to the screen.
